# CodeAlpha Data Analytics Internship

## Task 3: Data Visualization Using Python

### Project Overview

This project was completed as part of the CodeAlpha Data Analytics Internship Program.

The objective of this project is to analyze and visualize sales data using Python. Data visualization helps transform raw data into meaningful graphical representations, making it easier to identify trends, patterns, and business insights.

---

## Objective

The main goals of this project are:

* Load sales data from a CSV file.
* Perform basic data analysis.
* Create different types of visualizations.
* Generate business insights from the visualized data.

---

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Visual Studio Code (VS Code)

## Visualizations Created

### 1. Line Chart

Purpose:

* Shows the sales trend across months.
* Helps identify growth or decline in sales.

Output:

* line_chart.png

---

### 2. Bar Chart

Purpose:

* Compares sales values between months.
* Easily identifies high and low performing months.

Output:

* bar_chart.png

---

### 3. Pie Chart

Purpose:

* Displays the percentage contribution of each month to total sales.
* Helps understand distribution of sales.

Output:

* pie_chart.png

---

### 4. Histogram

Purpose:

* Shows the frequency distribution of sales values.
* Helps understand the spread of data.

Output:

* histogram_chart.png

---

## Key Insights

1. Sales show an overall increasing trend throughout the year.
2. December recorded the highest sales.
3. January recorded the lowest sales.
4. The last quarter (October–December) generated the highest revenue.
5. Business growth is visible throughout the year.

---

## Results

* Highest Sales: 45,000
* Lowest Sales: 12,000
* Best Performing Month: December
* Overall Trend: Increasing
